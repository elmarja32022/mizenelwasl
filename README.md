# ميزان الوصل ⚖️

## مذهب العدل والتبادل

منصة عالمية للتبادل العادل للخدمات والمنتجات - دعوة للبشرية جمعاء للعودة إلى الأصل.

---

## 🌟 الرؤية

**ميزان الوصل** ليس مجرد منصة رقمية، بل هو **مذهب حياة** يقوم على:

- **العدل** - أساس كل تعامل وتبادل
- **النية الحسنة** - جوهر كل عمل وقول
- **التكافل** - روح المجتمع الواحد
- **النزاهة** - عملة الثقة الحقيقية
- **الوصل والذكر** - الرابط مع الأصل

---

## ✨ المميزات

### 🔄 بنك الوقت
- كل خليفة يحصل على 5 ساعات عند التسجيل
- تبادل الخدمات بالوقت لا بالمال
- نظام عادل بدون ربا أو استغلال

### 🛡️ نظام النزاهة
- مقياس النزاهة وحسن النية
- مستويات الثقة: خليفة مميز، خليفة صادق، خليفة موثوق
- تقييم المجتمع الشفاف

### 📜 ميثاق الوصل والعهد
- 5 عهود ملزمة لكل خليفة
- التزام بالقيم والمبادئ
- لا حدود جغرافية • لا فوارق عنصرية

### 🌍 دعوة عالمية
- مفتوح للبشرية جمعاء
- 60+ دولة مدعومة
- الواجهة باللغة العربية (RTL)

---

## 🏗️ البنية التقنية

### الإطار الأساسي
- **Next.js 16** - App Router
- **TypeScript** - نوعية آمنة
- **Tailwind CSS** - تصميم متجاوب
- **Prisma** - ORM لقاعدة البيانات
- **SQLite** - قاعدة بيانات خفيفة

### المكتبات
- **Framer Motion** - حركات سلسة
- **shadcn/ui** - مكونات جميلة
- **Lucide React** - أيقونات
- **Radix UI** - وصولية عالية

## 📁 هيكل المشروع

```
src/
├── app/
│   ├── api/              # واجهات البرمجة
│   │   ├── auth/         # المصادقة (تسجيل/دخول/خروج)
│   │   ├── services/     # الخدمات
│   │   ├── products/     # المنتجات
│   │   ├── exchanges/    # التبادلات
│   │   ├── community/    # المجتمع
│   │   ├── notifications/ # الإشعارات
│   │   ├── categories/   # الفئات
│   │   ├── locations/    # المواقع
│   │   └── stats/        # الإحصائيات
│   ├── globals.css       # الأنماط العامة
│   ├── layout.tsx        # التخطيط الرئيسي
│   └── page.tsx          # الصفحة الرئيسية
├── components/
│   ├── ui/               # مكونات shadcn/ui
│   ├── auth/             # مكونات المصادقة
│   └── time-bank/        # مكونات بنك الوقت
├── hooks/                # React Hooks
└── lib/
    ├── db.ts             # اتصال Prisma
    ├── auth/             # أدوات المصادقة
    └── utils.ts          # أدوات مساعدة
```

## 🚀 التشغيل

```bash
# تثبيت المتطلبات
bun install

# توليد Prisma Client
bun run db:generate

# تشغيل الخادم التطويري
bun run dev

# بناء للإنتاج
bun run build

# تشغيل الإنتاج
bun run start
```

## 📊 API Endpoints

| Endpoint | الوصف |
|----------|-------|
| `POST /api/auth/register` | تسجيل خليفة جديد |
| `POST /api/auth/login` | تسجيل الدخول |
| `POST /api/auth/logout` | تسجيل الخروج |
| `GET /api/auth/me` | بيانات الخليفة الحالي |
| `GET/POST /api/services` | الخدمات |
| `GET/POST /api/products` | المنتجات |
| `GET/POST /api/exchanges` | التبادلات |
| `GET/POST /api/community` | المجتمع والمنشورات |
| `GET /api/stats` | الإحصائيات |
| `GET /api/categories` | الفئات |
| `GET /api/locations` | الدول والمدن |

## 📜 ميثاق الوصل والعهد

### العهود الخمسة:

1. **عهد النية والصدق** - النية أساس كل أفعالنا
2. **عهد الاستخلاف والبيئة** - راعٍ للأرض لا مستهلك
3. **عهد العدالة والتبادل** - لا ضرر ولا ضرار
4. **عهد العلم والابتكار** - العلم عبادة والابتكار خدمة
5. **عهد الوصل والذكر** - الرابط مع الأصل
